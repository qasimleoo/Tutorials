## 1. Conceptual Difference (High Level)

### Spring Framework

* Core framework hai
* Dependency Injection (DI) provide karta hai
* Aap ko **sab kuch manually configure** karna hota hai
* Web support built-in nahi hota, MVC alag se setup karna padta hai

Use case: Jab lightweight core features chahiyein

---

### Spring MVC

* Spring Framework ka **web module**
* Web applications aur REST APIs ke liye
* DispatcherServlet, Controllers, ViewResolvers sab manually configure
* XML ya Java config zyada hoti hai

Use case: Traditional web applications

---

### Spring Boot

* Spring Framework + Spring MVC ka **auto-configured version**
* Embedded server (Tomcat)
* XML almost zero
* Production-ready, fastest development

Use case: Microservices, REST APIs, modern apps

## A. Spring Framework (Core only)

### Configuration Class

```java
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class AppConfig {
    @Bean
    public HelloService helloService() {
        return new HelloService();
    }
}
```

### Service Class

```java
public class HelloService {
    public String message() {
        return "Hello Spring";
    }
}
```

### Main Class

```java
import org.springframework.context.ApplicationContext;
import org.springframework.context.annotation.AnnotationConfigApplicationContext;

public class MainApp {
    public static void main(String[] args) {
        ApplicationContext context =
                new AnnotationConfigApplicationContext(AppConfig.class);
        HelloService service = context.getBean(HelloService.class);
        System.out.println(service.message());
    }
}
```

Output console pe print hota hai
Web app nahi hai

## B. Spring MVC

### Controller

```java
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {
    @GetMapping("/hello")
    public String hello() {
        return "Hello Spring";
    }
}
```

### Web Initializer

```java
import org.springframework.web.servlet.support.AbstractAnnotationConfigDispatcherServletInitializer;

public class WebInit extends AbstractAnnotationConfigDispatcherServletInitializer {
    protected Class<?>[] getRootConfigClasses() {
        return null;
    }
    protected Class<?>[] getServletConfigClasses() {
        return new Class[]{WebConfig.class};
    }
    protected String[] getServletMappings() {
        return new String[]{"/"};
    }
}
```

### Config

```java
import org.springframework.context.annotation.ComponentScan;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.EnableWebMvc;

@Configuration
@EnableWebMvc
@ComponentScan
public class WebConfig {
}
```

Run on external Tomcat
URL:

```
http://localhost:8080/hello
```

## C. Spring Boot

### Application Class

```java
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

### Controller

```java
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {
    @GetMapping("/hello")
    public String hello() {
        return "Hello Spring";
    }
}
```

Run command:

```
mvn spring-boot:run
```

URL:

```
http://localhost:8080/hello
```

No XML
No external server
Minimum code

## 3. Side-by-Side Comparison

| Feature        | Spring   | Spring MVC | Spring Boot |
| -------------- | -------- | ---------- | ----------- |
| Web support    | No       | Yes        | Yes         |
| Configuration  | Manual   | Heavy      | Auto        |
| Server         | N/A      | External   | Embedded    |
| XML            | Possible | Mostly     | Rare        |
| Learning curve | Medium   | High       | Low         |
| Speed          | Slow     | Medium     | Fast        |

---

## Simple Summary

* **Spring** = Base engine
* **Spring MVC** = Web framework
* **Spring Boot** = Ready-made car with engine + web + fuel
