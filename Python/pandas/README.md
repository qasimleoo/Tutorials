## Intro

Pandas is an open src-python lib.. for data manipulation and analysis.

You can use Pandas to:

- see who bought the most
- add a new column: Did they tip?
- Save your customer list to a file
- find out the average number of lemonades bought

And you don't nee to count with fingers-

Pandas does it with just one line of code!

#### Installation:

`pip install pandas`

#### Import:

`import pandas as pd`

## Key data structures:

- Series = one list (single column table)
- DataFrames = Full table
