Intro?
What is Apchae server?
also know as httpd .. is an open-src web server that helps deliver the web pages to users over the internet.. 
IT processes requests from browseres and serves websites using the http protocol..

for example.. you have an html file .. you can test it locally but to test it on the internet for anyone .. you can use apache web server (httpd)
HTTPD ... stands for.. http request over a daemon server .. that runs in background ...
